History of CS: GO

Counter Strike (CS1.5, CS1.6): The earliest and most famous FPS game in game history, which inspired all the modern FPS games. CS was originally a mod in Half: Life, and established by Valve. Earliest game tournaments and prize tool is set in CS.

Counter Strike: Condition Zero: This version is like a middle transition from cs to cs:s, so the graphic and details inproved a lot for developing for 4 years.
